Project 1 - Get Away

Get Away, is a dynamic book finding app that allows the user to input a books name and be presented with important information about that book. Not only will you be show useful information about the book like the Author, page count, book rating, as well as a brief summary about the book. Lastly you will be provided with the current market value of the book and where you can purchase it. This app was made by Blake (myself), Jacob, and Cade as a team.
